This is the public repository for my Minecraft LED Clock project.

The project tutorial itself is available on Instructables (link to follow).

# Setting up
Clone (or download) this repository, and open the folder in VS Code with PlatformIO extension.
The dependencies of the projects will be automatically downloaded and the project will compile.

[Here is the documentation](https://docs.platformio.org/en/latest/integration/ide/vscode.html) on how to setup PlatformIO.
